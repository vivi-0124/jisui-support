## 📱 技術スタック

- **フレームワーク**: Next.js 15.3.4 (App Router + API Routes)
- **言語**: TypeScript 5, React 19
- **UI**: Tailwind CSS, Shadcn/ui, Lucide React
- **認証・DB**: Supabase
- **外部API**: Gemini AI, YouTube Data API
- **デプロイ**: Vercel
